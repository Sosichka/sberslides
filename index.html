Технологии

React

  • компонентный подход для построения удобного API
  • читабельность кода, его дальнейшее переиспользование
  • + бонус — разработка мобильного приложения на React.Native

Typescript

  • строгая типизация (тип any почти не используем)
  • избегаем «глупых» ошибок при разработке
  • читабельность кода
